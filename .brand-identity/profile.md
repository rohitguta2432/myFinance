## Glassmorphism Pattern

Used extensively for overlays, fixed bars, and premium elements:
```
bg-[color]/[opacity] backdrop-blur-[size] border border-white/[opacity]
```
Examples:
- **Fixed nav:** `bg-background-dark/80 backdrop-blur-lg`
- **Modal backdrop:** `bg-black/70 backdrop-blur-sm`
- **Premium card:** `bg-surface-dark/40 backdrop-blur-2xl border border-white/10`
- **Toast:** `background: rgba(15, 23, 42, 0.85), backdropFilter: blur(16px)`

## Anti-Patterns

Things this project explicitly avoids:

- No `bg-gradient-to-br from-purple-600 to-blue-500` or any generic AI gradients
- No `shadow-xl` on regular cards (use `shadow-sm` or border-only)
- No `bg-white` or `bg-gray-*` backgrounds (use semantic tokens: `bg-surface-dark`, `bg-background-dark`)
- No `text-gray-*` (use `text-slate-*` mapped through CSS variables)
- No `rounded-2xl` on standard cards (use `rounded-xl`)
- No arbitrary Tailwind values like `w-[347px]` or `mt-[13px]`
- No inline `style={{ }}` for colors (use Tailwind classes with CSS variable tokens)
- No `text-base` for body text (project uses `text-sm` as default)
- No new font families (Inter only)
- No CSS modules or styled-components (Tailwind only)
- No `dark:` Tailwind prefix (dark mode handled via CSS variables)
